# `@zphyrx/token-rc1`

## Installation

To install `@zphyrx/token-rc1`, run the following command:

```sh
# Using npm
$ npm install @zphyrx/token-rc1

# or using pnpm
$ pnpm add @zphyrx/token-rc1
```
